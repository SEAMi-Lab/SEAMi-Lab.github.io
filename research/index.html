研究方向/Research
智能人机交互技术
Research

聚焦婴幼儿视觉发育智能评估、大学生心理智能感知等前沿场景，探索人类与智能系统的高效交互模式。

Focuses on cutting-edge scenarios like intelligent assessment of infants' visual development and intelligent perception of college students' psychology, exploring efficient interaction modes between humans and intelligent systems.

用户体验驱动的软件工程
User Experience-Driven Software Engineering

围绕软件用户心智度量评价、用户体验评估模型等核心内容，持续提升软件产品的用户友好度与实用价值。

Centers on core content such as the measurement and evaluation of users' mental models for software and user experience assessment models, continuously enhancing the user-friendliness and practical value of software products.

融合式具身智能系统
Integrated Embodied Intelligent Systems

涵盖多智能体协同系统、无人机智能软件系统等方向，推动智能系统在实际场景中的高效落地与应用。

Encompasses directions including multi-agent collaborative systems and intelligent software systems for drones, promoting the efficient implementation and application of intelligent systems in real-world scenarios.

大语言模型应用技术
Large Language Model Application Technology

深耕合成生物智能体、大模型能力评测、数字辅导员等研究，不断拓展大语言模型的应用边界与产业价值。

Conducts in-depth research on synthetic biological agents, large model capability evaluation, and digital tutors, constantly expanding the application boundaries and industrial value of large language models.
